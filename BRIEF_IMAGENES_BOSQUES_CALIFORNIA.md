# Creative Brief: Requerimientos Fotográficos y Visuales
## Proyecto: Bosques California

Este documento detalla todas las piezas visuales necesarias para la actualización del sitio web y materiales de marketing. 

---

### 1. Identidad y Assets Base
*Imágenes que definen la marca en todo el ecosistema digital.*

*   [ ] **Logotipo Principal:** Formatos `.svg` y `.png` (transparente). Versión en blanco y versión Navy/Gold.
*   [ ] **Favicon / Isotipo:** Versión simplificada para navegadores.
*   [ ] **Loader de Sitio:** Imagen o icono para la pantalla de carga inicial.

---

### 2. Sección HERO (Página de Inicio)
*Impacto visual inmediato y alta resolución.*

*   [ ] **Imagen Hero Principal (Desktop):** Fachada exterior de gran impacto. (Recomendado: 2500px ancho, formato panorámico).
*   [ ] **Imagen Hero (Mobile):** Adaptación vertical de la fachada principal. (Recomendado: 1080x1920px).

---

### 3. Modelos Arquitectónicos (Casas)
*Cada modelo es un producto independiente y requiere su propio set visual.*

#### **Modelo: Roble A**
*   [ ] Fachada principal (Exterior).
*   [ ] Área de Sala (Interior).
*   [ ] Cocina / Comedor (Interior).
*   [ ] Recámara Principal.
*   [ ] Plano de Planta (2D o 3D Render).

#### **Modelo: Roble B**
*   [ ] Fachada principal (Ángulo alterno).
*   [ ] Sala de estar / Estancia.
*   [ ] Detalle de acabados en Cocina.
*   [ ] Baño principal o Vestidor.
*   [ ] Plano de Planta (2D o 3D Render).

#### **Modelo: Secuoya (Elite)**
*   [ ] Fachada principal (Frente imponente).
*   [ ] Estancia de doble altura (si aplica).
*   [ ] Terraza social con Deck.
*   [ ] Master Suite de lujo.
*   [ ] Plano de Planta (2D o 3D Render).

---

### 4. Amenidades y Estilo de Vida
*Venta de la experiencia "comunidad" y "lujo".*

*   [ ] **Cancha de Pádel:** Foto de la cancha en excelentes condiciones.
*   [ ] **Alberca Resort:** Toma abierta que incluya camastros y áreas de sombra.
*   [ ] **Gimnasio:** Enfoque en equipo moderno y amplitud.
*   [ ] **Terraza para Eventos:** Espacio social con mobiliario premium.
*   [ ] **Dog Park:** Área verde limpia y segura.
*   [ ] **Acceso Controlado:** Imagen que proyecte seguridad y tecnología.

---

### 5. Galería "Entorno Visual"
*Imágenes complementarias para rellenos dinámicos y masonry grids.*

*   [ ] **Detalles Constructivos:** (3 fotos) Texturas de piedra, madera, mármol.
*   [ ] **Paisajismo:** (2 fotos) Jardines, fuentes o senderos del desarrollo.
*   [ ] **Lifestyle:** (3 fotos) Detalles de una copa de vino, iluminación nocturna, o ambiente familiar.
*   [ ] **Vista Dron:** Toma aérea general del desarrollo y su entorno.

---

### Especificaciones Técnicas para Diseño
*   **Formatos Preferidos:** `.webp` (para web), `.jpg` (alta calidad), `.tiff` (RAW para edición).
*   **Exportación:** Todas las imágenes deben entregarse con un perfil de color sRGB.
*   **Nomenclatura Sugerida:** `bosques-california-seccion-nombre-modelo.ext` (ej: `bc-modelo-roble-a-fachada.jpg`).

---
**Nota:** El objetivo es lograr una estética "Cinematográfica y Premium" que iguale la experiencia de navegación del sitio actual.
